• élément

Énoncé

Faire en sorte qu'en cliquant sur le bouton ajouter, s'ajoute un élément à la liste.
Et que le bouton 'X' supprime l'élément <li> le contenant. (En bonus on peut précéder la suppression d'une animation.)

Remarque : vous pouvez adapter le html en ajoutant id et/ou class pour cibler les éléments.
